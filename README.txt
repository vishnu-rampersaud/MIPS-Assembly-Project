MIPS Assembly Project 
Author: Vishnu Rampersaud 
Completion Date: 3/26/2019


This repository contains an asm file of MIPS assembly code. 
The task consisted of drawing a symmetric blue cross onto a yellow background, contingent upon the variables m and n. 
This project was developed using the MARS MIPS simulator. 

Please note that this was a class project and a late upload
